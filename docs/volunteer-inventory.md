# Инвентарь волонтёра (админка)

Раздел инвентаря находится в карточке волонтёра на вкладке "Инвентарь".

## Функционал

### Просмотр инвентаря

Отображается таблица с предметами, которые находятся у волонтёра:

- **Предмет** — название предмета
- **Склад** — название склада, откуда был выдан предмет
- **Ячейка** — ячейка хранения
- **Кол-во** — количество предметов

### Передача предметов между волонтёрами

Кнопка **"Передать из личного инвентаря"** открывает модалку для передачи предметов от текущего пользователя другому волонтёру:

- **От кого** — текущий пользователь (автоматически заполняется, readonly)
- **Кому** — выбор волонтёра-получателя (поиск по базе)
- **Предмет** — выбор предмета из инвентаря отправителя
- **Кол-во** — количество для передачи (не больше доступного)

### Выдача предметов со склада

Кнопка **"Выдать со склада"** открывает модалку для выдачи предметов напрямую со склада:

- **Склад** — выбор склада (выпадающий список с поиском)
- **Предмет** — выбор предмета из доступных на выбранном складе (показываются только предметы с количеством > 0)
- **Кому** — текущий волонтёр (readonly)
- **Кол-во** — количество для выдачи (не больше доступного на складе)
- **Заметки** — опциональное поле для комментариев

После успешной выдачи:

- Модалка закрывается автоматически
- Инвентарь волонтёра перезагружается
- Показывается уведомление об успехе

## Технические детали

### Компоненты

- `InventorySection` — основной компонент секции инвентаря
- `InventoryTable` — таблица инвентаря
- `InventoryTransferModal` — модалка передачи между волонтёрами
- `StorageIssueModal` — модалка выдачи со склада
- `HistoryTab` — вкладка истории операций

### Хуки

- `useVolunteerInventory` — загрузка инвентаря волонтёра
- `useInventoryTransfer` — логика передачи предметов
- `useStorageIssue` — логика выдачи со склада

### API endpoints

- `GET /volunteer-inventory/{volunteerId}` — получение инвентаря волонтёра
- `GET /storage-positions/{positionId}` — получение информации о позиции склада
- `POST /storage-movements/` — создание перемещения между волонтёрами
- `GET /storage-positions/?storage={id}&has_count=true` — получение позиций склада (только с наличием)
- `POST /storage-positions/{positionId}/issue/` — выдача предмета со склада
