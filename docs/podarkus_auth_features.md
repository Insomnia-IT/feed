# Podarkus.ru - Процесс авторизации и функции для авторизованного пользователя

## 🔐 Процесс авторизации (2026)

### Метод 1: Авторизация по номеру телефона (основной)

1. Нажать кнопку **"Войти"** в верхней панели
2. Ввести номер телефона в формате `+7 (___) ___-__-__`
3. Нажать кнопку **"Войти"**
4. Получить SMS-код
5. Ввести код подтверждения
6. ✅ Авторизация успешна

### Метод 2: Авторизация по email (альтернативный)

1. Нажать кнопку **"Войти"** в верхней панели
2. Нажать кнопку **"Еще"** ✨ для показа альтернативных методов
3. Выбрать **"Электронная почта"**
4. Ввести email
5. Ввести пароль
6. Нажать кнопку **"Войти"**
7. ✅ Авторизация успешна

### Метод 3: Авторизация через социальные сети

1. Нажать кнопку **"Войти"** в верхней панели
2. Нажать кнопку **"Еще"** ✨
3. Выбрать социальную сеть в разделе **"Войти с помощью"**:
    - VK (ВКонтакте)
    - Telegram
    - Одноклассники
4. Подтвердить доступ в выбранной социальной сети
5. ✅ Авторизация успешна

---

## 📱 Навигация после авторизации

После успешной авторизации в верхней панели появляется:

- Иконка профиля / аватар
- Выпадающее меню с опциями:
    - **Мои вишлисты**
    - **Профиль**
    - **Настройки**
    - **Выйти**

## 🎁 Функции для авторизованного пользователя

### 1. Управление вишлистами

#### Создание вишлиста

- Нажать кнопку **"Создать вишлист"**
- Заполнить информацию:
    - Название вишлиста
    - Описание (опционально)
    - Обложка (изображение)
- Настроить видимость:
    - Публичный (доступен по ссылке)
    - Приватный (только для друзей)

#### Добавление товаров

- **Из каталога**: Кнопка **"Добавить в вишлист"** на карточке товара
- **Из интернет-магазина**:
    - Через браузерное расширение
    - Через форму добавления по URL
- **Вручную**:
    - Название товара
    - Цена
    - Ссылка
    - Изображение
    - Комментарий

#### Управление вишлистом

- Редактирование информации
- Добавление/удаление товаров
- Изменение порядка товаров
- Управление видимостью
- Статистика просмотров
- Удаление вишлиста

### 2. Друзья и социальные функции

#### Добавление друзей

- Поиск по email или имени
- Отправка приглашения по ссылке
- Импорт из социальных сетей
- QR-код для быстрого добавления

#### Взаимодействие с друзьями

- Просмотр вишлистов друзей
- Комментарии к товарам
- Личные сообщения
- Уведомления о событиях

### 3. Бронирование подарков

#### Процесс бронирования

1. Открыть вишлист друга
2. Выбрать товар
3. Нажать **"Забронировать"**
4. Указать комментарий (опционально)
5. Подтвердить бронирование

#### Управление бронированиями

- Просмотр активных бронирований
- Отмена бронирования
- История бронирований
- Статус покупки

### 4. Пригласительные открытки

- Создание цифровых приглашений
- Интеграция с вишлистами
- Отправка друзьям
- Трекинг ответов

### 5. Уведомления

#### Типы уведомлений

- 📦 О бронировании подарков
- 👥 Новые друзья
- 💬 Комментарии и сообщения
- 🎉 События в вишлистах
- 📢 Рекламные рассылки

#### Настройка уведомлений

- Email уведомления
- Push уведомления (в приложении)
- SMS уведомления (опционально)

### 6. Настройки профиля

#### Личные данные

- Имя и фамилия
- Дата рождения
- Email
- Номер телефона
- Аватар

#### Безопасность

- Смена пароля
- Двухфакторная аутентификация
- Активные сессии
- Привязанные социальные сети

#### Приватность

- Видимость профиля
- Видимость вишлистов
- Кто может добавлять в друзья
- Кто может бронировать подарки

## 📊 Статистика пользователя

После авторизации доступна личная статистика:

- **Создано вишлистов**: количество активных и архивных
- **Добавлено подарков**: общее количество товаров в вишлистах
- **Друзей**: количество подключенных друзей
- **Забронировано**: количество забронированных подарков
- **Получено подарков**: количество отмеченных как полученные

## ⚠️ Важные замечания

1. **Номер телефона** - основной способ авторизации. Рекомендуется привязать номер к аккаунту для удобства входа.

2. **Верификация email** - при регистрации через email может потребоваться подтверждение адреса.

3. **Безопасность** - не рекомендуется использовать одинаковые пароли на разных сайтах.

4. **Приватность** - тщательно настройте параметры приватности для контроля доступа к вашим вишлистам.
