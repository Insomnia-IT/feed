# Mind Map: Сервис вишлистов «Подаркус»

## Общее описание

**Подаркус** — бесплатный онлайн-сервис для создания вишлистов (списков подарков и желаний) с функцией бронирования подарков друзьями. Работает с 2018 года, более 3.5 млн пользователей.

## Структура страниц сайта

```
podarkus.ru/
├── / (Главная - каталог подарков)
├── /about (О сервисе)
├── /faq (Частые вопросы)
├── /certificates (Сертификаты)
│   ├── /all (Полный каталог сертификатов)
│   ├── /all/{category} (Категории)
│   └── /view/preview (Пример сертификата)
├── /articles (Статьи и подборки)
│   └── /{article-slug} (Конкретная статья)
├── /store/{id} (Страница магазина)
│   └── /product/{id} (Страница товара)
├── /profile (Личный кабинет)
├── /wishlist/{id} (Публичный вишлист)
├── /b2b/certificates (Добавить сертификат для партнеров)
├── /privacy-policy (Политика конфиденциальности)
└── /terms-of-service (Пользовательское соглашение)
```

## Ключевые возможности

### Вишлисты

- Неограниченное количество списков желаний
- Добавление товаров из любых интернет-магазинов по ссылке
- Парсинг информации о товаре (название, цена, изображение)
- Категоризация подарков
- Публичные и приватные списки
- Совместное использование для событий

### Социальные функции

- Добавление друзей
- Просмотр вишлистов друзей
- Бронирование подарков (чтобы не дублировались)
- Анонимность бронирования для дарителя

### Сертификаты

- Электронные подарочные сертификаты
- Широкий выбор категорий (массаж, красота, спорт, образование и др.)
- Гибкая сумма (от 2 000 до 300 000 руб.)
- Мгновенная отправка получателю по SMS
- Активация по QR-коду
- Возможность использования частями

### Мобильность

- PWA (Progressive Web App)
- Нативные приложения для iOS и Android
- Мобильная версия сайта

## Бизнес-модель

1. **Реклама и партнерские программы** — интеграция с интернет-магазинами
2. **Продажа сертификатов** — комиссия от партнеров
3. **B2B-направление** — добавление сертификатов в каталог
