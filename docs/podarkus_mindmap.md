## 🎯 Ключевые функции

### 1. **Создание вишлиста**

- Добавление товаров из любых интернет-магазинов
- Ручное добавление желаний
- Настройка видимости списка
- Публикация и обмен ссылкой

### 2. **Добавление друзей**

- Приглашение друзей по ссылке
- Просмотр вишлистов друзей
- Бронирование подарков для друзей

### 3. **Бронирование подарков**

- Бронирование товаров из вишлистов
- Отслеживание статуса бронирования
- Предотвращение дублирования подарков

### 4. **Каталог подарков**

- Готовые подборки товаров
- Интеграция с интернет-магазинами
- Сортировка и фильтрация
- Быстрое добавление в вишлист

### 5. **Мобильные приложения**

- iOS приложение (App Store)
- Android приложение (Google Play)
- Синхронизация с веб-версией

## 🔐 Авторизация

### Способы входа:

1. **Номер телефона** (основной способ)
    - Вход по SMS-коду
    - Регистрация по номеру
2. **Электронная почта**
    - Email + пароль
    - Восстановление пароля

3. **Социальные сети**
    - VK
    - Telegram
    - Одноклассники
