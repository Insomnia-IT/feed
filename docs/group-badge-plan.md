## План по групповым бейджам

Экран "План по групповым бейджам" отображает информацию о планируемом количестве питаний по каждому групповому бейджу на текущий приём пищи.

### Расположение

Экран доступен из раздела "История" через кнопку "План по групповым бейджам", которая находится под кнопкой "Статистика кормлений".

### Структура данных

На экране отображается таблица с четырьмя колонками:

1. **Название** — название группового бейджа
2. **Служба** — название службы (direction), к которой относится групповой бейдж
3. **План** — планируемое количество питаний на текущий приём пищи
4. **Факт** — фактическое количество выданных питаний на текущий приём пищи

### Логика расчёта плана

План для каждого группового бейджа рассчитывается следующим образом:

1. Система ищет ячейку планирования (`MealPlanCell`) для текущего приёма пищи (`mealTime`) и текущей даты
2. Если ячейка найдена:
    - План = `amount_meat + amount_vegan` (сумма мясных и веганских порций)
3. Если ячейка не найдена:
    - План = количество волонтёров с этим групповым бейджем, находящимся на поле

### Архитектура компонента

Компонент состоит из следующих частей:

- **`use-group-badge-plan-stats.ts`** — хук, который получает данные о групповых бейджах и волонтёрах, рассчитывает план для каждого бейджа
- **`group-badge-plan.tsx`** — основной компонент экрана
- **`group-badge-plan-table.tsx`** — компонент таблицы для отображения данных
- **`group-badge-plan-screen.tsx`** — экран с заголовком и навигацией

### Фильтрация данных

Отображаются только групповые бейджи, которые:

- Не отключены (`is_disabled = false`)
- Не удалены (`deleted_at = null`)
- Принадлежат текущей кухне или являются общими (`kitchen = null`)

### Пример использования

```typescript
import { useGroupBadgePlanStats } from 'shared/hooks/use-group-badge-plan-stats';

const { stats } = useGroupBadgePlanStats();
// stats: Array<{ id: number; name: string; direction: string | null; plan: number; fact: number }>
```
